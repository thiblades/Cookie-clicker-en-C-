# Party Clicker (Cookie Clicker en C#)

Party Clicker est un projet en réponse à un challange Let's Dev (Association de Centrale Lille Formation IG2I).

**Challenge accepté**

Party Clicker est un jeu de type Cookie Clicker centré sur un théme festif :tada: codé en C#
avec la bibliothéque SFML.

## Commencer

### Prérequis

Le projet est conçu a partir du SDK .NET Core 2.0.

Les dépendances et assemblys suivantes sont requises afin de faire fonctionner le projet:

- Newtonsoft.Json ([Disponible sur NuGet](https://www.nuget.org/packages/Newtonsoft.Json))
- SFML.Net 2.2 ([Assemblys ici](https://www.sfml-dev.org/download/sfml.net/))

### Installer

Un projet Visual Studio est fourni afin de vous permettre de compiler et d'installer le projet très facilement.
Il vous suffit d'ouvrir le projet et de générer la solution.

Sinon, vous pouvez aussi utiliser Microsoft Build Tools directement `MsBuild.exe Clicker.sln` .

## Lancer

Le projet compilé est disponible dans le dossier `bin`.

Vous pouvez l'executer directement.

## Flemme de compiler

On vous comprend, on vous prépare une archive pour que vous n'ayez rien à faire
